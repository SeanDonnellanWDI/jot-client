Q&A

What is Jot

Jot is a note taking app. Jot allows users to write down their ideas and easily organize them later via a drag and drop system.


Who is the target audience?

Jot was originally created for songwriters in mind. Rearranging lyrics and verses using copy and paste methods can be annoying, so a simple drag and drop solution is much easier. Additionally, Jot is a fantastic list maker, perfect for busy users who simply need to write down all their to-do list tasks and order them later.


How does Jot work?

Currently, a users can sign up (Join) then sign in (Login) using real or fake credentials, or users can use these provided example credentials to Login:

u: jot@example.com
p: jot

When a user successfully signs in, that user can change their password (please do not edit the example account password) or sign out. A signed-in user will also see the "Write a jot" section (left) where they can create new jots, and they can see the "Read your jots" section (right) where they can read, edit, or delete their existing jots.

Jot often prompts the user with success and error messages as well which informs the user if something has or has not happened, such as successfully creating a jot, or failing to sign up because the username was already taken. These user messages appear above the "write a jot" section.


What's next for Jot?

Jot is still in its infancy right now and new challenges are being tackled everyday. The next features to be developed are the drag and drop feature, jot-grouping feature, social sharing platform, and the final UX style guide.
